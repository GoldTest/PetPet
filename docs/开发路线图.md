# PocPet 开发路线图

这份路线图面向玩家，说明 PocPet 接下来想做的方向。实际发布时间和功能细节可能会根据测试反馈、平台适配和存档兼容调整。

## 我们坚持的方向

- 保持轻松陪伴，不做强制签到和失败惩罚。
- 离开几天也不会被重罚，回来时更偏向欢迎和轻量补给。
- 让每天打开都有一点变化，但不把游戏变成任务清单。
- 新内容优先服务陪伴感、收集感和长期记忆。
- 保护已有存档和 Mod，不轻易破坏玩家已经养成的宠物。

## 当前版本：1.1.0

1.1.0 主要补强“长期陪伴基础”和日常体验细节。

已纳入的重点：

- 每日小愿望：每天刷新一个轻量目标，例如喂食、清洁、玩耍、摸摸或打工一次。
- 回归欢迎：离开较久后触发恢复任务和补给，不做强惩罚。
- 番茄钟体验优化：计时区域更清楚，轮次用箭头增减，操作更适合桌面和移动端。
- 天气和季节信息优化：保留氛围信息，但减少界面拥挤。
- 照顾反馈优化：极低饱腹、低清洁睡觉等状态会有更明确提示。
- 背包和商店可读性优化：更容易看懂道具会影响哪些状态。

## 下一阶段：1.2

1.2 会继续增强长期陪伴、收集和外观目标。以下内容是计划方向，不代表一次更新会全部完成。

计划方向：

- 纪念册 / 回忆册：记录生日、节日、年度总结、第一次升级、第一次获得小心心、番茄钟累计等重要回忆。
- 成就系统第一版：提供轻量长期目标，不做连续签到惩罚，也不让奖励破坏经济平衡。
- 小窝 / 房间装扮：先从背景和少量摆件开始，让宠物生活空间更有个人痕迹。
- 宠物饰品：围巾、铃铛、帽子、睡帽、节日装饰等永久外观。
- 种植 / 花园：种植普通树苗，随真实时间成长，成熟后产出水果道具；忘记照顾不会枯死。
- 宠物偏好扩展：根据长期互动形成更个性化的反馈文本、每日小愿望和轻量奖励。
- Mod 导入体验增强：更清楚地提示 Mod 文件问题、缺失资源和兼容性情况。

## 更远一点的想法

这些内容还在探索中，会在基础系统稳定后逐步评估：

- 宠物性格标签：根据长期喂食、清洁、互动、番茄钟等行为形成倾向。
- 记忆文本：宠物会提到最近或长期发生过的事情。
- 季节活动扩展：春夏秋冬限定小事件、限定装饰和纪念物。
- 更完整的收集图鉴：记录纪念物、节日物品、装饰和事件。
- 更完整的外观与房间编辑：在不增加压力的前提下，提供更多可收集、可展示内容。

## 暂时不会做的事

- 不做高压连续签到。
- 不做离线失败惩罚。
- 不做联网排行、强竞争或强付费压力。
- 不允许 Mod 注入脚本或直接修改核心数值。
- 不为了新增玩法牺牲旧存档兼容。

## 反馈方向

如果你愿意反馈，最有帮助的是：

- 哪些日常操作最舒服，哪些太打扰。
- 哪些道具、奖励或提示看不懂。
- 你希望宠物记住哪些事情。
- 你更想先看到小窝、成就、纪念册、饰品、花园中的哪一类内容。
